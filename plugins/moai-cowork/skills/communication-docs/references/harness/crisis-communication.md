# 위기 커뮤니케이션 (86-crisis-communication)

> MoAI-Cowork v0.2.0 하네스 레퍼런스

## 개요

위기 상황 분석, 대응 전략 수립, 메시지 개발, 커뮤니케이션 실행을 수행하는 에이전트 팀 협업 하네스입니다.

## 전문가 역할

- **위기 분석가**: 위기 상황을 분석합니다.
  - 상황: 위기 상황 파악
  - 영향: 영향 범위 및 정도
  - 이해관계자: 주요 이해관계자
  - 우려: 주요 우려사항
  - 위험: 추가 위험 요소

- **전략가**: 대응 전략을 수립합니다.
  - 목표: 위기 관리 목표
  - 접근: 커뮤니케이션 접근
  - 타이밍: 대응 타이밍
  - 채널: 커뮤니케이션 채널
  - 조정: 조정 및 동기화

- **메시지 개발자**: 효과적인 메시지를 개발합니다.
  - 핵심: 핵심 메시지
  - 투명성: 솔직하고 투명한 표현
  - 공감: 공감과 배려 표현
  - 행동: 취해질 행동
  - 정보: 추가 정보 제공

- **실행 관리자**: 커뮤니케이션을 실행합니다.
  - 계획: 실행 계획
  - 일정: 실행 일정
  - 책임: 역할 배치
  - 모니터링: 반응 모니터링
  - 조정: 필요한 조정

## 워크플로우

### 단계 1: 준비

1. 사용자 입력에서 추출:
   - **위기**: 위기 상황
   - **영향**: 영향받는 대상
   - **목표**: 관리 목표
   - **제약**: 제약 조건
   - **자원**: 이용 가능한 자원

2. `_workspace/` 디렉토리 생성

### 단계 2: 팀 구성 및 실행

| 순서 | 작업 | 담당 | 의존성 | 산출물 |
|------|------|------|--------|--------|
| 1 | 분석 | 분석가 | 없음 | `01_crisis_analysis.md` |
| 2 | 전략 | 전략가 | 작업 1 | `02_response_strategy.md` |
| 3 | 메시지 | 개발자 | 작업 2 | `03_messaging.md` |
| 4 | 실행 | 관리자 | 작업 3 | `04_execution_plan.md` |

## 산출물

- `01_crisis_analysis.md` — 위기 분석
- `02_response_strategy.md` — 대응 전략
- `03_messaging.md` — 메시지 개발
- `04_execution_plan.md` — 실행 계획

## 오류 처리

| 오류 유형 | 대응 전략 |
|---------|---------|
| 정보 부족 | 수집 가능 범위 내 진행 |
| 신속성 필요 | 우선순위 조정 |
| 에이전트 실패 | 1회 재시도 → 진행 |
