# 데이터 분석 (87-data-analysis)

> MoAI-Cowork v0.2.0 하네스 레퍼런스

## 개요

데이터 수집, 정제, 분석, 시각화, 인사이트 도출을 수행하는 에이전트 팀 협업 하네스입니다.

## 전문가 역할

- **데이터 엔지니어**: 데이터를 수집하고 준비합니다.
  - 수집: 다양한 소스에서 데이터 수집
  - 정제: 데이터 오류 및 중복 제거
  - 표준화: 데이터 형식 표준화
  - 통합: 여러 데이터 소스 통합
  - 검증: 데이터 품질 검증

- **데이터 분석가**: 데이터를 분석합니다.
  - 탐색: 기초 통계 분석
  - 패턴: 주요 패턴 파악
  - 상관관계: 변수 간 관계 분석
  - 추세: 추세 및 변화 분석
  - 가설: 가설 검정

- **시각화 전문가**: 데이터를 시각화합니다.
  - 차트: 적절한 차트 선택
  - 설계: 효과적인 시각화 설계
  - 상호작용: 상호작용식 시각화
  - 명확성: 명확하고 이해하기 쉬운 표현
  - 미학: 전문적인 외관

- **인사이트 제공자**: 인사이트를 도출합니다.
  - 해석: 분석 결과 해석
  - 의미: 비즈니스 의미 도출
  - 권고: 행동 권고
  - 우선순위: 우선순위 결정
  - 경고: 주의해야 할 점

## 워크플로우

### 단계 1: 준비

1. 사용자 입력에서 추출:
   - **데이터**: 분석할 데이터 또는 소스
   - **질문**: 답해야 할 질문
   - **목표**: 분석 목표
   - **제약**: 데이터 제약
   - **기한**: 분석 기한

2. `_workspace/` 디렉토리 생성

### 단계 2: 팀 구성 및 실행

| 순서 | 작업 | 담당 | 의존성 | 산출물 |
|------|------|------|--------|--------|
| 1 | 수집 | 엔지니어 | 없음 | `01_data_preparation.md` |
| 2 | 분석 | 분석가 | 작업 1 | `02_analysis_report.md` |
| 3 | 시각화 | 전문가 | 작업 2 | `03_visualizations.md` |
| 4 | 인사이트 | 제공자 | 작업 3 | `04_insights_recommendations.md` |

## 산출물

- `01_data_preparation.md` — 데이터 준비
- `02_analysis_report.md` — 분석 보고서
- `03_visualizations.md` — 시각화
- `04_insights_recommendations.md` — 인사이트 및 권고
