# 세무 도우미 (Tax Helper)

> moai-finance | 세무 전문 스킬

## 실행 모듈

### 세무 도우미 (3.3%/홈택스)
한국 세법 기반 세무 상담. 3.3% 원천징수, 종합소득세, 부가가치세, 홈택스 안내.
- 가이드: `references/tax/guide.md`
- 참조: `references/tax-helper/guide.md`

## 트리거 키워드

세금, 세무, 부가세, 3.3%, 원천징수, 종합소득세, 종소세, 홈택스, 사업자 세금, 연말정산, 절세

## 실행 규칙

1. 사용자 요청 수신 → 세무 유형 판별
2. `references/tax/guide.md` 로드 → 한국 세법 기반 분석
3. **세무/재무 판단은 항상** `mcp__sequential-thinking__sequentialthinking` 호출
4. 결과물 생성 후 사용자 검토 요청

⚠️ 면책 고지: AI 어시스턴트이며 공인 세무/회계 자문을 대체하지 않습니다. 중요한 세무 사안은 반드시 세무사와 상담하세요.

## 사용 예시

- "이번 달 프리랜서 수입 300만원인데 3.3% 계산해줘"
- "종합소득세 신고 기간이랑 방법 알려줘"
- "개인사업자 부가가치세 신고 어떻게 해?"
- "홈택스에서 원천징수 영수증 어떻게 발급해?"
- "프리랜서인데 절세 방법 있어?"

## 문제 해결

- **세율 변경 확인 불가**: 세율은 매년 변경될 수 있습니다. 국세청 홈택스(hometax.go.kr) 또는 세법 개정 고시를 직접 확인하세요.
- **과거 데이터 미제공**: 종합소득세 신고 지원 시 전년도 소득·경비 내역이 필요합니다. 홈택스에서 소득 자료 조회 후 제공해 주세요.
- **지방세 연동 계산**: 지방소득세(소득세의 10%)는 별도 신고가 필요하며 위택스(wetax.go.kr)에서 처리합니다.
- **원천징수의무자 불분명**: 사업소득(3.3%)과 근로소득(연말정산)이 혼재된 경우 세무사와 소득 분류를 먼저 정리하세요.

## 이 스킬을 사용하지 말아야 할 때

- **실제 세금 신고·납부**: 홈택스 신고 대행은 공인 세무사에게 의뢰하세요. AI가 작성한 내용은 참고용입니다.
- **법인세 신고**: 법인 결산 및 법인세 신고는 `close-management` 스킬과 세무사를 활용하세요.
- **재무제표 작성**: K-IFRS 기준 재무제표 작성은 `financial-statements` 스킬을 사용하세요.
- **세무 조사 대응**: 세무조사 대응은 반드시 세무사·변호사와 함께 진행하세요.
