# 협력업체 견적서 세부 원가 분석

## 역할
협력업체 견적서를 재료비·가공비·노무비·경비·물류비·관리비·이윤으로 분해해 적정성, 업체 간 차이, 협상 포인트를 도출한다.

## 입력 데이터
- 견적서 파일: XLSX/CSV/PDF 표 추출본
- 업체명, 품번, 품명, 규격, 수량, 단가, 금액
- 세부 원가: 재료비, 가공비, 노무비, 제조경비, 금형/치공구, 포장, 물류, 일반관리비, 이윤
- 기준 가격: target price, 전년 단가, 최저 견적, 내부 BOM 단가
- 조건: MOQ, 납기, Incoterms, 환율, 유효기간

## 워크플로우
1. 데이터 표준화: 업체명·품번·통화·단위·수량 기준 정규화
2. 원가 구조 분해: 세부 항목별 비중과 단위당 원가 계산
3. 업체 간 비교: 동일 품목 기준 최저/최고/평균 단가, gap, 순위 산출
4. 이상치 탐지: 항목별 비중, 단가, 이윤율, 물류비, 금형비 급등 항목 표시
5. target gap 분석: target price 대비 차이액/차이율, 협상 우선순위 도출
6. 협상 포인트 작성: 가격 인하 근거, 대체 업체, 물량 조정, 조건 변경 제안

## 계산식
- 총 견적 단가 = 재료비 + 가공비 + 노무비 + 제조경비 + 포장/물류 + 일반관리비 + 이윤
- 항목 비중 = 항목 금액 / 총 견적 금액
- target gap = 견적 단가 - target price
- gap rate = target gap / target price
- 업체 간 gap = 업체 단가 - 동일 품목 최저 단가

## 산출물
- 협력업체별 견적 비교표
- 품목별 원가 구성 waterfall
- target price gap 목록
- 이상치/검증 필요 항목
- 협상 우선순위 Top 10
- 구매 협상 질문 리스트

## 사용하지 말아야 할 때
- 내부 BOM roll-up이 목적이면 `bom-cost-rollup` 사용
- 공장별 제조원가 비교가 목적이면 `factory-cost-comparison` 사용
