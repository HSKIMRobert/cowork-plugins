# 전체 프로젝트 원가 2025/2026 비교

## 역할
프로젝트 전체 원가를 2025/2026 기준으로 비교해 총원가 증감, 제품별 기여도, 공장/협력업체/BOM 영향, 수익성 변화를 통합 분석한다.

## 입력 데이터
- 프로젝트명, 고객, 제품, 공장, 협력업체
- 연도/월: 2025, 2026
- 판매량, 매출, ASP, 총원가, 재료비, 노무비, 제조경비, 물류비, 개발비, 금형비
- BOM roll-up 결과, 협력업체 견적 분석 결과, 공장 원가 분석 결과
- 예산/견적/실적 구분, 환율, 생산지 변경, 사양 변경 이력

## 워크플로우
1. 데이터 매핑: 제품·공장·협력업체·BOM·프로젝트 코드 연결
2. 총원가 비교: 2025 vs 2026 총액, 단위당 원가, 원가율 산출
3. 기여도 분석: 제품별·공장별·협력업체별 증감 기여도 계산
4. 원가 bridge: 단가, 수량, BOM 변경, 생산지 변경, 환율, 물류, 불량 영향 분해
5. 수익성 분석: 매출총이익, 원가율, 프로젝트 마진 변화 계산
6. 액션 플랜: 절감 과제, 협상 과제, 설계 변경, 생산지 최적화 우선순위 도출

## 계산식
- 프로젝트 총원가 = Σ 제품별 총원가
- 단위당 프로젝트 원가 = 프로젝트 총원가 / 판매량
- 원가율 = 프로젝트 총원가 / 매출
- 마진 = 매출 - 프로젝트 총원가
- 마진율 = 마진 / 매출
- 증감 기여도 = 항목별 증감액 / 프로젝트 총 증감액

## 산출물
- 프로젝트 2025/2026 원가 비교 요약
- 제품별/공장별/협력업체별 증감 기여도
- 원가 bridge 표
- 수익성/마진 변화 분석
- 원가 절감 과제 우선순위
- 경영진 요약 보고서 초안
