# 내부 생산공장 원가 비교

## 역할
공장별 제조원가 자료를 제품·연도 기준으로 비교해 생산지별 원가 경쟁력, 증감 원인, 개선 대상을 찾는다.

## 입력 데이터
- 공장, 제품 코드, 제품명, 프로젝트, 고객
- 연도/월: 2025, 2026
- 생산량, 양품 수량, 불량률, 가동률
- 재료비, 직접노무비, 제조간접비, 감가상각, 에너지비, 물류비, 외주가공비
- 표준원가, 실제원가, 예산원가

## 워크플로우
1. 기준 통일: 통화, 단위, 제품 코드, 기간, 생산량 기준 표준화
2. 단위당 원가 계산: 총원가 / 생산량, 양품 기준 원가 별도 산출
3. 2025/2026 비교: 제품별·공장별 증감액/증감률 계산
4. 공장 간 비교: 동일 제품 기준 공장별 원가 차이와 순위 산출
5. 차이 원인 분해: 재료비, 노무비, 간접비, 물류비, 불량률, 가동률 영향 분리
6. 개선 후보 도출: 고원가 공장, 급등 제품, 생산이관 검토 대상 식별

## 계산식
- 단위당 제조원가 = 총 제조원가 / 생산량
- 양품 기준 원가 = 총 제조원가 / 양품 수량
- 공장 gap = 해당 공장 단위당 원가 - 동일 제품 최저 공장 원가
- 연도 gap = 2026 단위당 원가 - 2025 단위당 원가
- 불량비 영향 = 양품 기준 원가 - 생산량 기준 원가

## 산출물
- 제품별/공장별 2025/2026 원가 비교표
- 공장별 원가 경쟁력 순위
- 원가 증감 bridge
- 고원가 제품/공장 Top 10
- 불량률·가동률 영향 분석
- 생산지 전환 검토 후보

## 사용하지 말아야 할 때
- BOM 부품 단가 중심이면 `bom-cost-rollup` 사용
- 프로젝트 전체 원가/마진 비교이면 `project-cost-variance` 사용
